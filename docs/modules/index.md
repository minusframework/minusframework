# Documentação Técnica dos Módulos

Cada módulo do MinusFrameWork é versionado e publicado como repositório independente.
Abaixo estão os links para a documentação técnica interna de cada um.

| Módulo | Descrição | Repositório |
|--------|-----------|-------------|
| [Core](Core/) | Núcleo compartilhado (conexão, atributos, tipos) | [minusframework-core](https://github.com/GabrielFerreiraMendes/minusframework-core) |
| [ORM](ORM/) | Repositório genérico, queries, mapeamento | [minusframework-orm](https://github.com/GabrielFerreiraMendes/minusframework-orm) |
| [Migrator](Migrator/) | Migração de schema, CLI e DLL | [minusframework-migrator](https://github.com/GabrielFerreiraMendes/minusframework-migrator) |
| [Messaging](Messaging/) | Message bus, sagas, outbox | [minusframework-messaging](https://github.com/GabrielFerreiraMendes/minusframework-messaging) |
| [FeatureFlags](FeatureFlags/) | Feature toggles, SSE, REST API | [minusframework-featureflags](https://github.com/GabrielFerreiraMendes/minusframework-featureflags) |
| [Extensions](Extensions/) | Horse, JWT, integrações | [minusframework-extensions](https://github.com/GabrielFerreiraMendes/minusframework-extensions) |
| [Telemetry](Telemetry/) | Tracing, logging estruturado | [minusframework-telemetry](https://github.com/GabrielFerreiraMendes/minusframework-telemetry) |
| [AI](AI/) | MCP Server, agentes inteligentes | [minusframework-ai](https://github.com/GabrielFerreiraMendes/minusframework-ai) |
