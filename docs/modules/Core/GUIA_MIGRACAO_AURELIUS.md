## Visao Geral

| Conceito | Aurelius | EntityDAC | **MinusORM** |
|---|---|---|---|
| Atributos de mapeamento | `[Entity]`, `[Column]`, `[Id]` | `[Entity]`, `[Column]`, `[Key]` | `[Tabela]`, `[Coluna]`, `[ChavePrimaria]` |
| Contexto / Session | `TMSAureliusConnection` + `TMSAureliusManager` | `TEntityDACConnection` + `TEntityContext` | `IConexao` + `TRepositorioBase<T>` |
| Query fluent | `TCriteria<T>` | `TLinqQuery<T>` | `TConstrutorSelecao<T>` + `TCriteria` |
| Migrations | Manual SQL ou Third-party | `TEntityDACMigration` | **MinusMigrator** integrado |
| Idiomas | Ingles | Ingles | **Portugues** (primario) + **aliases em ingles** |
| Licenca | Commercial | Commercial | **MIT (Community) + Commercial (Enterprise)** |

## Mapeamento de Atributos

### Tabela / Entity

```pascal
// Aurelius
[Entity]
[Table('PRODUTOS')]
TProduto = class

// EntityDAC
[Entity]
[Table('PRODUTOS')]
TProduto = class

// MinusORM
[Tabela('PRODUTOS')]
TProduto = class
```

### Colunas

```pascal
// Aurelius
[Column('NOME', [TColumnProp.Required], 100)]
property Nome: string read FNome write FNome;

// EntityDAC
[Column('NOME')]
[Required]
[StringLength(100)]
property Nome: string read FNome write FNome;

// MinusORM
[Coluna('NOME')]
[Obrigatorio]
[TamanhoMaximo(100)]
property Nome: string read FNome write FNome;
```

### Chave Primaria

```pascal
// Aurelius
[Id('FId', TIdGenerator.IdentityOrSequence)]
property Id: Integer read FId write FId;

// EntityDAC
[Key]
[AutoGenerated]
property Id: Integer read FId write FId;

// MinusORM
[ChavePrimaria]  // auto-detecta estrategia pelo provider
property Id: Integer read FId write FId;
```

### Atributos Comuns â€” Tabela de Equivalencia

| Aurelius | EntityDAC | **MinusORM** |
|---|---|---|
| `[Id]` | `[Key]` | `[ChavePrimaria]` |
| `[Column('X')]` | `[Column('X')]` | `[Coluna('X')]` |
| `[Table('X')]` | `[Table('X')]` | `[Tabela('X')]` |
| `[TColumnProp.Required]` | `[Required]` | `[Obrigatorio]` |
| `[TColumnProp.Unique]` | â€” | `[ChaveUnica]` |
| `[StringLength(N)]` | `[StringLength(N)]` | `[TamanhoMaximo(N)]` |
| `[Association]` | `[Association]` | `[Relacionamento]` |
| `[Enumeration]` | `[Enumeration]` | `[ConversorTipo]` |
| `[Transient]` | `[NotMapped]` | `[Ignorar]` |
| â€” | â€” | `[Sombra]` (Shadow Properties) |
| â€” | â€” | `[Criptografado]` |
| â€” | â€” | `[CriadoPor]` / `[CriadoEm]` |
| â€” | â€” | `[AtualizadoPor]` / `[AtualizadoEm]` |
| â€” | â€” | `[Versao]` (lock otimista) |
| â€” | â€” | `[SoftDelete]` |
| â€” | â€” | `[Inquilino]` (multi-tenancy) |

## Mapeamento de Classes e Interfaces

```pascal
// Aurelius
var
  LManager: TMSAureliusManager;
  LConnection: IDBConnection;
begin
  LConnection := TMSAureliusConnection.CreateConnection(...);
  LManager := TMSAureliusManager.Create(LConnection);

// MinusORM
var
  LConexao: IConexao;
  LRepo: IRepositorioBase<TProduto>;
begin
  LConexao := TRegistroProvedores.CriarConexao(LParams);
  LConexao.Conectar(LParams);
  LRepo := TRepositorioBase<TProduto>.Create(LConexao);
```

## CRUD Operations

```pascal
// Aurelius
LManager.Find<TProduto>(42);
LManager.FindAll<TProduto>;
LManager.Save(LProduto);
LManager.Remove(LProduto);
LManager.Flush;

// MinusORM
LRepo.BuscarPorId(42);
LRepo.BuscarTodos;
LRepo.Salvar(LProduto);
LRepo.Excluir(LProduto.Id);

// MinusORM â€” Fluent (alternativa)
LProduto := ORM_Selecionar<TProduto>.PorId(42).BuscarUnico;
LProdutos := ORM_Selecionar<TProduto>.Listar;
ORM_Inserir('PRODUTO').Values.Add('NOME', 'X').Executar;
ORM_Atualizar(LConexao, 'PRODUTO').Values.Add('NOME', 'Y').Where.Add(Criterio('ID').Igual(1)).Executar;
ORM_Excluir(LConexao, 'PRODUTO').Where.Add(Criterio('ID').Igual(1)).Executar;
```

## Query API

```pascal
// Aurelius Criteria
LManager.Find<TProduto>
  .CreateAlias('Categoria', 'c')
  .Add(TExpression.Eq('Ativo', True))
  .Add(TExpression.GreaterThan('Preco', 100))
  .SetMaxResults(10)
  .List;

// MinusORM SelectBuilder
TConstrutorSelecao<TProduto>
  .Criar
  .Include('Categoria')
  .Onde(E([Criterio('ATIVO').Igual(True),
            Criterio('PRECO').MaiorQue(100)]))
  .Pular(0).Pegar(10)
  .Listar;

// MinusORM Expression Type-Safe
var LCond := TCond<TProduto>.Criar;
var LPreco := LCond.Campo<Currency>('PRECO');
var LAtivo := LCond.Campo<Boolean>('ATIVO');
LRepo.Buscar(LAtivo = True, LPreco > 100);
```

## Transactions e UnitOfWork

```pascal
// Aurelius
LManager.Connection.BeginTransaction;
try
  LManager.Save(LProduto);
  LManager.Flush;
  LManager.Connection.Commit;
except
  LManager.Connection.Rollback;
end;

// MinusORM â€” Transaction explicita
var LTx := LConexao.IniciarTransacao;
try
  LRepo.Salvar(LProduto);
  LTx.Confirmar;
except
  LTx.Reverter;
end;

// MinusORM â€” UnitOfWork (recomendado)
var LUoW := TUnidadeTrabalho.Create(LConexao);
LUoW.RegistrarNovo(LProduto);
LUoW.Confirmar;
```

| Paginacao | Manual | Manual | `TPaginacao` + `TResultadoPaginado<T>` |

| Async | Manual | Manual | `SalvarAsync`, `BuscarPorIdAsync`, `ParaCadaAsync` |

## Checklist de Migracao

### Fase 1 â€” Renomear Atributos (automated, ~30 min)

- [ ] Substituir `[Entity]` por `[Tabela]`
- [ ] Substituir `[Table('X')]` por `[Tabela('X')]`
- [ ] Substituir `[Id]` / `[Key]` por `[ChavePrimaria]`
- [ ] Substituir `[Column('X')]` por `[Coluna('X')]`
- [ ] Substituir `[Transient]` / `[NotMapped]` por `[Ignorar]`
- [ ] Substituir outros atributos conforme tabela de equivalencia

### Fase 2 â€” Adaptar Contexto/Session (~1h)

- [ ] Substituir `TMSAureliusConnection` / `TEntityDACConnection` por `IConexao`
- [ ] Substituir `TMSAureliusManager` / `TEntityContext` por `TRepositorioBase<T>`
- [ ] Adaptar parametros de conexao para `TParametrosConexao`

### Fase 3 â€” Adaptar CRUD (~2h)

- [ ] Substituir `Find<T>(Id)` por `BuscarPorId`
- [ ] Substituir `FindAll<T>` por `BuscarTodos`
- [ ] Substituir `Save` / `Update` por `Salvar`
- [ ] Substituir `Remove` / `Delete` por `Excluir`

### Fase 4 â€” Adaptar Queries (~3h)

- [ ] Substituir Criteria Aurelius por `TConstrutorSelecao<T>` + `Criterio()`
- [ ] Substituir LINQ EntityDAC por Expression Type-Safe
- [ ] Atualizar joins, includes, eager loading para `Include()` / `[Relacionamento]`

### Fase 5 â€” Migracoes de Schema (~1h)

- [ ] Configurar `MinusMigrator_CLI.exe` no projeto
- [ ] Rodar `minusmigrator init` e `minusmigrator add-migration InitialCreate`
- [ ] Configurar CI/CD com `minusmigrator migrate --force`

### Fase 6 â€” Testes e Validacao (~2h)

- [ ] Rodar suite de testes existente
- [ ] Validar queries geradas com `TProfiler`
- [ ] Validar concorrencia com `[Versao]`
- [ ] Testar rollback com `ITransactionScope`

## Script de Renomeacao em Massa (PowerShell)

```powershell
# rename-attributes.ps1 â€” Execute na raiz do projeto
$files = Get-ChildItem -Recurse -Filter *.pas

foreach ($f in $files) {
  $content = Get-Content $f.FullName -Raw
  $changed = $false

  $replacements = @{
    '[Entity]'                            = '[Tabela]'
    '\[Table\('                          = '[Tabela('
    '\[Id\]'                             = '[ChavePrimaria]'
    '\[Id\('                            = '[ChavePrimaria('
    '\[Key\]'                            = '[ChavePrimaria]'
    '\[Key\('                           = '[ChavePrimaria('
    '\[Column\('                       = '[Coluna('
    '\[Transient\]'                       = '[Ignorar]'
    '\[NotMapped\]'                       = '[Ignorar]'
    '\[Required\]'                        = '[Obrigatorio]'
    '\[StringLength\('                  = '[TamanhoMaximo('
    '\[Association\('                   = '[Relacionamento('
    '\[Version\]'                         = '[Versao]'
    'TMSAureliusManager'                  = 'TRepositorioBase'
    'TEntityContext'                      = 'TRepositorioBase'
    'TMSAureliusConnection'               = 'IConexao'
    'TEntityDACConnection'                = 'IConexao'
  }

  foreach ($key in $replacements.Keys) {
    if ($content -match $key) {
      $content = $content -replace $key, $replacements[$key]
      $changed = $true
    }
  }

  if ($changed) {
    Write-Host "Atualizado: $($f.Name)" -ForegroundColor Yellow
    Set-Content $f.FullName -Value $content
  }
}

Write-Host "Migracao de atributos concluida! Revise as alteracoes e compile." -ForegroundColor Green
```
