# MinusExtensions

Wrappers e bridges para bibliotecas de terceiros do ecossistema Delphi.

## Bibliotecas Inclusas

- Horse (HTTP framework)
- Horse CORS, JWT, Logger Lumberjack (middlewares)
- Jhonson (JSON middleware)
- Delphi Jose JWT (JOSE/JWT)
