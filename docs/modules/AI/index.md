# MinusAI

Servidor MCP (Model Context Protocol) e agentes inteligentes para aplicações Delphi.

## Recursos

- **MCP Server** — integração com LLMs via protocolo padronizado
- **Agentes** — execução de tarefas com chain-of-thought
- **Ferramentas** — expõe funções do seu sistema como tools para a IA
