## Produtos Derivados (Planejado)

- **MinusOutbox** — Transactional Outbox Pattern integrado ao ORM
- **MinusQuery** — Tradutor OData/GraphQL para criteria API
- **MinusFixture** — Motor de geração de dados para testes
- **MinusHealth Dashboard** — Painel de monitoramento em tempo real
- **MCP Server** — Model Context Protocol para integração com IAs
- **minuSCM** — Sistema de Controle Comercial (prova de conceito)
- **minusERP** — ERP completo rodando com o framework
